# Teste_PySpark_Docker_Postgree
Teste_PySpark_Docker_Postgree

Para o Spark 3.3 e Iceberg 1.6.1, você precisará dos seguintes JARs:
Iceberg Spark Runtime: iceberg-spark-runtime-3.3_2.12-1.6.1.jar

PostgreSQL JDBC Driver: postgresql-42.7.3.jar (ou a versão mais recente compatível)
Estes arquivos devem ser baixados e colocados no diretório ./jars.
